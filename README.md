# PORTFOLIO

## 制作概要
WordPressオリジナルテーマで構築したポートフォリオサイトです。  
アニメーションやUI/UXにこだわり、「視認性」と「没入感」を意識して制作しました。

---

## 担当範囲
- デザインカンプをもとにコーディング
- WordPressオリジナルテーマ化
- レスポンシブ対応
- アニメーション実装
- UI / UX調整
- 表示速度を意識した実装

---

## 使用技術
- WordPress
- PHP
- HTML
- Sass
- JavaScript
- GSAP
- ScrollTrigger
- Swiper
- Gulp

---

## 工夫した点
- ローディングからMVへのアニメーション演出
- ScrollTriggerを使用したインタラクション実装
- モバイル時の可読性・操作性の最適化
- Sassを用いた保守性を意識した設計
- 表示速度やアニメーション負荷を考慮した実装
